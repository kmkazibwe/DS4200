Social Media Data Analysis with D3.js

This website explores how social-media engagement can be understood through data visualization. By collecting and analyzing metrics such as likes and shares, we can visualize trends across different age groups, platforms, and time periods. Using D3.js, I’ve built interactive visualizations that make it easier to see how people interact online and how behavior changes over time.

This project demonstrates the power of combining coding, analytics, and storytelling to bring data to life. Beyond this project, I’m passionate about how visualization helps simplify complex information, making it more accessible for everyone. I hope this work inspires others to explore the clarity and creativity that thoughtful data design can offer.
